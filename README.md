# Python Template
## Basics
### Stack
- Python
- FastAPI
- MongoDB (via ODMantic)
- RabbitMQ (soon)
- Poetry
- Linters (flake8, wemake-style-guide, mypy)
- Formatters (black, autoflake)
- Pre-commit
- Pytest

Версии можно найти в [pyproject.toml](./pyproject.toml)

### Install
```
pip install poetry
poetry install
pre-commit install
```
